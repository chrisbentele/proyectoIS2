Detailed Description

Registro de modelos de datos.

En este archivo se definirán los modelos correspondientes a los datos de todo el sistema.
Entro ellos se encuentran los siguientes:

  • Usuario: El modelo correspondiente a cualquier usuario del sistema.
  • Proyecto: Modelo correspondiente a un proyecto.
